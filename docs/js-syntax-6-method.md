## メソッド
- オブジェクトのプロパティとなった関数のこと

### メソッドの宣言
```javascript
let obj = {
  a: function(x) {return typeof x} // 従来の書き方
  b: y => typeof y // ES2015から
  c:(z) {return typeof z} // ES2015から
}
```
